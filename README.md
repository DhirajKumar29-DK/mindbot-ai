# MindBot AI

MindBot AI is a full-stack AI chatbot built using React, Node.js, Express, and OpenRouter API.

It provides intelligent real-time conversations with multiple AI personalities through a clean and interactive user interface.

## Features

* Real-time AI-powered chat responses
* Multiple AI personalities
* Dark mode support
* Message timestamps
* Clear chat functionality
* Responsive and user-friendly interface

## Tech Stack

**Frontend**
React, Vite, CSS

**Backend**
Node.js, Express.js

**API Integration**
OpenRouter API

## Project Highlights

* Built a complete full-stack AI chatbot from scratch
* Integrated external AI API for dynamic responses
* Implemented personality-based prompt engineering
* Designed a modern and interactive chat interface
